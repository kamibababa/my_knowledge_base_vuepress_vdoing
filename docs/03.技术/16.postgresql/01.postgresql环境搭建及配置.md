### 安装

官网下载最新稳定版，傻瓜式安装

```
postgresql-16.6-3-windows-x64.exe
```

安装时，会要求设置超级管理员用户（postgre)的密码，我设置成了root

### 配置环境变量

编辑Path环境变量：

在“环境变量”窗口中，找到系统变量中的“Path”变量，然后点击“编辑(E)...”按钮。
添加新条目：

在“编辑环境变量”窗口中，点击“新建(N)”按钮。
在弹出的输入框中，输入PostgreSQL的bin目录路径：C:\Program Files\PostgreSQL\16\bin。
